# Tugas 1 - Fitur Pengembalian Buku

## Deskripsi
Mengimplementasikan fitur pengembalian buku pada Sistem Informasi Perpustakaan menggunakan Laravel.

## Fitur
- Menampilkan detail transaksi.
- Tombol **Kembalikan Buku** pada halaman detail.
- Mengubah status transaksi menjadi **Dikembalikan**.
- Menghitung denda keterlambatan sebesar **Rp5.000 per hari**.
- Menampilkan total denda.
- Menambah stok buku sebanyak **1** setelah buku dikembalikan.

## File yang Diubah
- `app/Http/Controllers/TransaksiController.php`
- `app/Models/Transaksi.php`
- `resources/views/transaksi/index.blade.php`
- `resources/views/transaksi/show.blade.php`
- `routes/web.php`
